Пример автотеста с использованием selenium с использованием модели PageObject, работающий с фреймворком pytest.
В тесте предусмотрен вариант автоматический генерации вносимых данных. Чтобы внести данные контакта вручную необходимо 
в файле main_page.py закомментировать блок кода "данные контакта (вариант с генерацией)" и раскомментировать блок "данные контакта"
Для запуска требуется установить необходимые пакеты и запустить тест, поочерёдно вызвав команды в консоли. 

pip install -r requirements\requirements.txt
pytest
